Picker Photo (sent on picker_send)

Upload a file OR paste a URL, then click Apply Photo. ✕ clears it. If empty, [photo] markers in your script are skipped (use video instead).

GIF (sent on [gif] markers)

Upload a .gif (or paste a URL) — it'll animate inside the chat bubble. Use [gif] in easy format on a line like Mom: [gif], or gif_in: GIF / gif_out: GIF in advanced format.

Picker Video (sent on video markers)

Upload a poster image (first-frame screenshot of your video). The bubble shows a play-button overlay so it looks like a video. Use [video] in easy format, or video_in: VIDEO / video_out: VIDEO in advanced format.

Easy Script (paste & play)

Two formats accepted (auto-detected per textarea):
1) Easy: Speaker: message per line. date: TEXT = date separator. Pick the POV speaker above.
Media inline: click + 📷 / + 🎞 / + 🎬 above a textarea — your file is uploaded into the in-page media library and a marker is inserted at the cursor. Markers: [gif:NAME], [photo:NAME], [video:NAME]. Bare [gif] / [photo] / [video] still use the single-slot uploads in the sections above.
2) Advanced (timed): paste lines like [1.5] type: hello + [0.8] send for outgoing, [2] typing: 2 + [0.8] in: text for incoming, [1] picker_open / picker_tap 1 / picker_send for photos. POV dropdown is ignored — direction is implicit. The script is passed through verbatim (no auto "Today" header).
Advanced script (auto-generated — edit only if you know the syntax)
Script syntax:
[seconds] action: content

POV (your side):
• type: text — types in input with key flashes
• backspace: N — deletes N chars
• send — sends what's in input
• clear_input — wipes input
• out: text — instant outgoing bubble
• image_out: URL — send image directly
• gif_out / gif_out: GIF — send your uploaded GIF

Photo picker (POV — most realistic):
• picker_open — opens inline Photos picker above keyboard
• picker_tap N — taps photo N (1 = Bloom AI hero, 2-16 = camera roll)
• picker_send — sends the selected photo (closes picker)
• picker_close — dismisses without sending

Their side:
• typing: N — typing N seconds
• typing_stop — typing disappears
• in: text — incoming
• image_in: URL — incoming image
• gif_in / gif_in: GIF — incoming animated GIF
• video_in: VIDEO / video_out: VIDEO — video bubble (poster + play button + duration)

Status:
• delivered — "Delivered" under last sent
• read or read TIME — "Read" with timestamp

Other:
• reaction_in_last: TYPE / reaction_out_last: TYPE
types: heart, like, dislike, haha, emphasis, question
• date: TEXT
• call_in: N
• hide_keyboard / show_keyboard
• wait: N / clear
